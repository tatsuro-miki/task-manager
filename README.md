# AIタスク管理ツール

> 話しかけるだけで完結する、月額数十円のAIタスク管理

## 概要

食品輸入卸（創業160年超）の1人情シス担当が、多岐にわたる業務タスクの管理漏れを解消するために内製したAIチャット型タスク管理ツール。

| 項目 | 内容 |
|------|------|
| 対象 | 食品輸入卸・1人情シス担当 |
| 課題 | タスクを記憶で管理しており、チェック漏れ・対応忘れが発生しやすい状態 |
| 解決 | Claude APIによるチャット型タスク管理ツールをVercel上に構築 |
| 効果 | 自然な日本語で話しかけるだけでタスク管理が完結。チェック漏れをゼロに |
| コスト | Claude API使用料のみ（1日10回で月約20〜45円） |

## 使用技術

- HTML + JavaScript
- Vercel Serverless Functions
- Claude API（Anthropic）
- Google Apps Script（データ永続化）

## 主な機能

| 機能 | 内容 |
|------|------|
| 自然言語タスク登録 | 「明日までにZohoベンダーへ連絡、優先度高で追加」のように話しかけるだけで登録 |
| 期日・優先度の自動判定 | 「明日」「今日中」「来週」等の表現を自動解釈 |
| リマインダー通知 | 期日当日・1日前・期限切れを自動検知して通知 |
| データ永続化 | Google Apps Script経由でスプレッドシートに保存。複数デバイスから参照可能 |
| 一括操作 | 「全部終わった」で一括完了。AIが既存タスクを自動判別して完了・編集を実行 |

## 操作例

```
「明日までにZohoベンダーへ連絡、優先度高で追加」
→ タスク追加（期日・優先度を自動設定）

「全部終わった」
→ すべてのタスクを一括完了

「Zohoの件のメモに来週担当者と電話確認と追加して」
→ 既存タスクを自動判別してメモに追記
```

## 既存SaaSとのコスト比較

既存のタスク管理SaaS（Notion・Asana等）のAI機能は有料プランでのみ利用可能で、ユーザー数に応じて月額費用が増加します。本ツールは**API使用料のみで運用可能なため、SaaSと比較して大幅なコスト削減**を実現しています。

| 比較観点 | 本ツール | Notion/Asana等 |
|----------|----------|----------------|
| 月額費用 | **API使用料のみ** | 無料〜数千円/ユーザー/月 |
| AI自然言語操作 | ○（チャットで完結） | △（有料プランのみ） |
| カスタマイズ性 | 業務フローに合わせて自由に改修可 | 範囲限定 |
| 社内データ管理 | Googleスプレッドシート（社内完結） | 外部クラウド保存 |

| 操作負荷 | ツールを開いて手動入力 | 話しかけるだけで完結 |
